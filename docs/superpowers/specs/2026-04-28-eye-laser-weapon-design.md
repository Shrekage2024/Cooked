# Eye Laser Weapon — Design Spec

**Date:** 2026-04-28  
**Status:** Approved

---

## Overview

Tim gains a third weapon: a rapid-burst eye laser mapped to the Space key. Pressing Space fires 3 small blue laser bolts from Tim's eye in his facing direction, staggered 100ms apart. This gives Tim a full-screen ranged option that complements the existing melee saw (A) and radial slam (S).

---

## Mechanic

- **Trigger:** Space key (JustDown, same pattern as A and S)
- **Burst:** 3 bolts fired 100ms apart (burst completes at 200ms)
- **Cooldown:** 3000ms starting after the third bolt fires
- **Speed:** 700 px/s in the facing direction
- **Damage:** 1 per bolt (up to 3 total if all three hit)
- **Piercing:** No — each bolt destroys itself on first enemy contact
- **Out-of-bounds:** Bolts are destroyed when they exit the screen

### Stat comparison

| Weapon     | Key   | Damage      | Range       | Cooldown |
|------------|-------|-------------|-------------|----------|
| Saw swing  | A     | 1 (AoE)     | Melee       | 500ms    |
| Hammer slam| S     | 2 (radial)  | Medium      | 1500ms   |
| Eye laser  | Space | 1×3 bolts   | Full screen | 3000ms   |

---

## Sprite

A single PNG at `assets/images/laser-bolt.png`:
- **Canvas size:** 64×20px
- **Orientation:** Horizontal (pointing right)
- **Visual:** Bright white core, fading to cyan/blue glow toward the edges — a tight, elongated ellipse
- **In-game display size:** ~40×14px (scaled down in code)
- Rotated 90° in code for up/down facing shots

The sprite is a static image (no animation sheet needed).

---

## Origin Point

Bolts spawn offset from Tim's center to approximate his eye position (top of his head, slightly forward):

| Facing | Offset (x, y) |
|--------|---------------|
| right  | (+30, −12)    |
| left   | (−30, −12)    |
| up     | (+8, −30)     |
| down   | (−8, +30)     |

## Implementation Scope

### New constants (`constants.js`)
- `LASER_DAMAGE = 1`
- `LASER_COOLDOWN = 3000`
- `LASER_SPEED = 700`
- `LASER_BURST_COUNT = 3`
- `LASER_BURST_INTERVAL = 100`

### New sprite (`assets/images/laser-bolt.png`)
Generated by a small Python/Pillow script — a 64×20 glowing blue bolt, white core.

### `Boot.js`
- Load `laser-bolt` as a static image (not a spritesheet)

### `Tim.js`
- Add `_laserCooldown` state and `_keySpace` binding
- Add `_handleLaser()` called from `update()`
- Add `_fireLaserBurst()` which uses `scene.time.addEvent` to fire 3 bolts 100ms apart
- Add `_spawnLaserBolt()` which creates a physics-enabled image with velocity set from facing direction, origin offset applied, and `worldBounds` destroy enabled

### `CollisionHandler.js`
- New overlap: `laserBolts` group vs `enemies` group
- On overlap: deal `LASER_DAMAGE` to enemy, destroy the bolt

### `Game.js`
- Create `laserBolts` physics group, pass to Tim constructor and CollisionHandler
- Enable `worldBounds` on the group so bolts auto-destroy on screen exit

---

## Out of Scope

- Laser sound effect (audio not yet wired up project-wide)
- Bolt animation / trail particles
- Bolt-vs-tree interaction (bolts pass through trees)
