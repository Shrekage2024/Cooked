# Eye Laser Weapon Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a Space-key rapid-burst eye laser to Tim — 3 blue bolts fired 100ms apart, traveling 700 px/s, each dealing 1 damage to the first enemy hit, with a 3000ms cooldown after the burst completes.

**Architecture:** A new `laserBolts` physics group lives on the Game scene and is passed into Tim (for spawning) and CollisionHandler (for overlap detection). Tim fires bolts via a staggered `time.addEvent` burst and auto-destroys them on world-boundary exit via Phaser's `worldbounds` physics event. The laser bolt sprite is a 64×20 PNG generated by a Python script.

**Tech Stack:** Phaser 3 Arcade Physics (ES6 modules), Python 3 + Pillow for sprite generation, `npx serve .` for local dev server.

## Task 1: Generate the laser bolt sprite

**Files:**
- Create: `assets/images/laser-bolt.png`

This task uses Python to generate a 64×20 PNG — a horizontal glowing bolt with a bright white core fading to cyan/blue at the edges. Run this from the project root.

- [ ] **Step 1: Verify Pillow is available**

```bash
python3 -c "from PIL import Image, ImageDraw, ImageFilter; print('ok')"
```

Expected: `ok`. If you see `ModuleNotFoundError`, run `pip3 install Pillow` first.

- [ ] **Step 2: Generate the sprite**

```bash
python3 - <<'EOF'
from PIL import Image, ImageDraw, ImageFilter
import math

W, H = 64, 20
img = Image.new("RGBA", (W, H), (0, 0, 0, 0))
draw = ImageDraw.Draw(img)

cx, cy = W / 2, H / 2

for x in range(W):
    for y in range(H):
        dx = (x - cx) / (W / 2)
        dy = (y - cy) / (H / 2)
        dist = math.sqrt(dx * dx + dy * dy)
        if dist > 1.0:
            continue
        t = 1.0 - dist
        # White core → cyan glow
        r = int(255 * t)
        g = int(255 * t)
        b = 255
        a = int(255 * (t ** 0.6))
        img.putpixel((x, y), (r, g, b, a))

# Soft blur to smooth the glow
img = img.filter(ImageFilter.GaussianBlur(radius=1))
img.save("assets/images/laser-bolt.png")
print("Saved assets/images/laser-bolt.png")
EOF
```

Expected output: `Saved assets/images/laser-bolt.png`

- [ ] **Step 3: Verify the file was created**


Expected: file exists, ~1–3 KB.

## Task 4: Add laserBolts group to Game.js

**Files:**
- Modify: `src/scenes/Game.js`

This task creates the `laserBolts` physics group on the scene, registers a `worldbounds` listener to auto-destroy bolts that leave the screen, and passes the group to Tim.

- [ ] **Step 1: Update `_buildGroups()` to create the laserBolts group**

Current `_buildGroups()`:
```js
  _buildGroups() {
    this.enemies = this.physics.add.group();
    this.attackHitboxes = this.physics.add.group();
  }
```

Replace with:
```js
  _buildGroups() {
    this.enemies = this.physics.add.group();
    this.attackHitboxes = this.physics.add.group();
    this.laserBolts = this.physics.add.group();

    // Destroy bolts that exit the screen
    this.physics.world.on('worldbounds', (body) => {
      if (body.gameObject && this.laserBolts.contains(body.gameObject)) {
        body.gameObject.destroy();
      }
    });
  }
```

- [ ] **Step 4: Add `_handleLaser()`, `_fireLaserBurst()`, and `_spawnLaserBolt()` methods**

Add these three methods after `_handleSlam()` (before `_spawnHitbox`):

```js
  _handleLaser() {
    if (this._laserCooldown > 0) return;
    if (!Phaser.Input.Keyboard.JustDown(this._keySpace)) return;
    this._laserCooldown = LASER_COOLDOWN + LASER_BURST_INTERVAL * (LASER_BURST_COUNT - 1);
    this._fireLaserBurst();
  }

  _fireLaserBurst() {
    this.scene.time.addEvent({
      delay: LASER_BURST_INTERVAL,
      repeat: LASER_BURST_COUNT - 1,
      callback: () => this._spawnLaserBolt(),
    });
  }

  _spawnLaserBolt() {
    const eyeOffsets = {
      right: [30, -12],
      left:  [-30, -12],
      up:    [8, -30],
      down:  [-8, 30],
    };
    const velocities = {
      right: [LASER_SPEED, 0],
      left:  [-LASER_SPEED, 0],
      up:    [0, -LASER_SPEED],
      down:  [0, LASER_SPEED],
    };
    const angles = { right: 0, left: 180, up: 270, down: 90 };

    const [ox, oy] = eyeOffsets[this._facing];
    const [vx, vy] = velocities[this._facing];

    const bolt = this.scene.physics.add.image(
      this.x + ox,
      this.y + oy,
      'laser-bolt',
    );
    bolt.setDisplaySize(40, 14);
    bolt.setAngle(angles[this._facing]);
    bolt.setDepth(DEPTH.HITBOX);
    bolt.damage = LASER_DAMAGE;
    bolt.body.setCollideWorldBounds(true);
    bolt.body.onWorldBounds = true;
    bolt.setVelocity(vx, vy);

    this._laserBolts.add(bolt);
  }
```
